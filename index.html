Eleuth P2P
Decentralized Messaging

Eleuth is an open-source peer to peer messaging solution based on connected rings and chains network. Eleuth goal is to provide enough infrastructure to create decentralized nework for messaging where there are firewalls and internet shutdowns.

Eleuth P2P Features

Open Source

We keep all the sources of Eleuth network open. Developers from all over the world can observe the process and help us.

Easy to Launch

We make installation and running process easy for everyone.

Secure

Security is our main purpose. We make sure no one else can read your data by encrypting it.

Anonymous

Eleuth does its best to keep users anonymous in the network by cutting off central authorization

Decentralized

There is no central governance in Eleuth network. No single authority can bring down everything.
